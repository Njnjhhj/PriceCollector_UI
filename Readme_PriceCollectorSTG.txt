Preparation

1. Create a folder 'PriceCollectorDataSTG' on disk C: .
2. Create inside of folder 'PriceCollectorDataSTG' 2 sub-folders: 'InputSTG' and 'OutputSTG'. 
3. Put xslx-file with name 'ProductIdSTG' into 'InputSTG' folder.
4. 'ProductIdSTG' file should contain only one column with header 'ProductId'. List of product IDs should be under the header.
5. Result of work should be in 'OutputSTG' sub-folder of 'PriceCollectorDataSTG' folder. File name format: OutputSTG_YYYYMMDD_HHmmss 

Using

1. Open MS Visual Studio.
2. Open 'PriceCollector' project.
3. Find 'PriceCollector.CollectorStgPro' in Solution Explorer (right panel in Visual Studio) and open it.
4. Find 'Executor_Stg.cs'. Double click.
5. There is 1 test method:	'StgPro_Collector'
6. Find annotation [Test] above the method.
7. Right click on the annotation > Left click on 'Run test/s'.
8. Wait for finis of the programm.
